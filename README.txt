get FIPS codes.py -
	This file takes in a string of the names of counties, and given the FIPS code
	for the state, prints out the respective FIPS codes for the counties listed.

error queries.sql -
	This file contains the SQL queries that pick out all possible error scenarios in the
	scraped and raw data.

runErrorQueries.py -
	This file runs each error query and prints out the output to the command line.
